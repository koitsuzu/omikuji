# 冰川神社御神籤 (Hikawa Shrine Omikuji)

這是一個致力於還原冰川神社參拜體驗的 Web 應用程式，現在支援完整 50 首籤詩的日中雙語切換。

## 版本功能總結
- **完整 50 首御神籤整合**：完整收錄冰川神社全系列 50 首籤詩，並包含特殊的「籤王」。
- **日中雙語即時切換**：
  - 在結果彈窗中新增「JP ↔ 中」切換按鈕。
  - 支援即時變更語系，無需重新載入或重新抽取，保持內容一致性。
- **極速流暢體驗**：所有翻譯數據均預載於 JavaScript 腳本中，實現零延遲的語言切換。
- **UI 交互與體驗優化**：
  - **修復點擊阻擋問題**：解決了硬幣動畫產生的透明層遮擋「結束奉納」按鈕導致無法繼續的問題。
  - **中文字型優化**：針對繁體中文排版進行了字體與行距調整，維持原有神社風格的優雅質感。
- **全流程參拜模擬**：包含洗手淨身、投幣、祈鈴、祈禱到抽籤的完整互動流程。

## 使用技術
- HTML5 / Vanilla CSS3 / JavaScript (ES6+)
- 靜態數據驅動架構

## 部署與運行
直接開啟 `index.html` 即可運行，或部署至 GitHub Pages 進行線上瀏覽。
